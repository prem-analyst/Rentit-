# 🚗 RentIt - Online Equipment Rental Management System

An automated web-based platform designed to simplify equipment rental operations, track inventory, and manage customer transactions efficiently. Developed as part of the final year BSc Computer Science curriculum at SPPU.

---

## 🛠️ System Architecture & Requirements

### 💻 Software Configuration
- **Operating System:** Windows 10 / 11
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend Environment:** Python / PHP
- **Database Management:** MySQL Server

### 🔌 Hardware Specification
- **Processor:** Intel Core i3 (or equivalent) and above
- **Memory:** 4 GB RAM minimum (8 GB Recommended)
- **Storage:** 20 GB available hard drive space

---

## 📊 Database Design & Modules

The system's backend is structured around core relational database modules to prevent redundancy and handle data smoothly:

- **User & Customer Management:** Handles registration, secure login credentials, and user profiles.
- **Inventory & Category Module:** Tracks equipment availability, specifications, rental rates, and categories.
- **Booking & Transaction System:** Manages dynamic booking constraints, rental durations, and invoice generation.

## 📊 Data Analytics Extension
To showcase data analyst capabilities, this project includes a business intelligence extension:
- **Dataset (`rental_data.csv`):** Generated and structured a mock database of 1,000+ transactional rental records.
- **SQL Business Insights (`insights.sql`):** Developed complex SQL queries to analyze key business metrics including Gross Revenue, Revenue Contribution by Category, Top Demand Products, Preferred Payment Methods, and Low-Rating Customer Feedback Alerts for operational maintenance.
